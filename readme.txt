Most advanced, highly customizable, flexible and easy-to-use card widget for Elementor with lots of options and possibilities.

== Description ==

*Create stunning, out of the box content card, image card, image box, content box, promotional product or anything that needs an image, title, description and a button. And customize every single piece the way you want. You have the control to make yourself happy.*

### Best Features

- Everything is customizable for every elements. For example, border, shadow, padding, margin etc.
- 3 image positions (Left, Top, Right, Bottom will be added if you want), means lots of possibilities.
- Option to choose image size or set a custom image size, you have the control.
- Highly customizable button like the default *button widget*.
- Very light weight base CSS means **fast and optimized loading**.
- 9 different label positions with out of the box possibilities.
- **SEO friendly markup**.
- Select heading level (h1-h6) based on content hierarchy, also best for SEO.
- 3 different content alignment options.
- Responsive settings control.
- More amazing features are on the way ...

== Installation ==

It's really easy and super simple to install **Happy Elementor Card** plugin but before installing **Happy Elementor Card** make sure you've installed [Elementor](https://wordpress.org/plugins/elementor/ "Install Elementor"). **Elementor** is the only dependency.

= Automatic Installation =
1.  Go to 'Plugins > Add New' screen in WordPress.
2.  Search for `Advanced Card Widget for Elementor`.
3.  Install and activate the plugin, that's it.

= Manual Installation =
1.  Download [Happy Elementor Card](https://downloads.wordpress.org/plugin/happy-elementor-card.zip "Download Happy Elementor Card").
2.  Extract the `happy-elementor-card.zip` file. You'll get plugin files inside `happy-elementor-card` directory.
3.  Upload the plugin files to the `/wp-content/plugins/happy-elementor-card` directory.
4.  Activate the plugin through the 'Plugins' screen in WordPress.
